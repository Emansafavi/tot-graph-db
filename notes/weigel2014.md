# Summary
This paper explores on-skin interaction and treats the body itself as an input surface for mobile computing. It shows that different skin locations afford different gestures and meanings, and that physical comfort or discomfort can be deliberately used as part of interaction design. It is valuable because it frames touch as bodily, anatomical, and affective rather than neutral.

# Key Concepts
- skin as input surface
- body-as-interface
- anatomical specificity
- bodily awareness
- affect in interaction

# Notes
This paper strongly suggests that skin is not just a surface but a differentiated interface with expressive and functional variation.

A very interesting point is that uncomfortable gestures can be reserved for irreversible or important actions.

Compared to many musical interface discussions, this work takes bodily meaning much more seriously.
